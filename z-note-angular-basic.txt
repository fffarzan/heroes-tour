>> Workspace:  workspace contains the files for one or more projects.

>> project: is the set of files that comprise an app, a library, or end-to-end (e2e)
tests.

>> creating a new workspace: ng new <workspace-name>
files that created in the workspace:
1-inital skeleton app project: src subfolder.
2-end-to-end test project: e2e subfolder.
3-configuration files.

>> serving application: ng serve --open 
    Starts the development server and watching the source files.
    --open flag opens a browser to http://localhost:4200/.
